Connections - CI/CD Flow
Source Code > Build Test

Build Test > SonarQube
Docker Images > ECR
Deploy > EKS Cluster
Terraform > AWS Cloud

// Traffic Flow
Route53 > CloudFront
CloudFront > WAF
WAF > Ingress Controller
Ingress Controller > Frontend Pod
Ingress Controller > Backend Pod
Ingress Controller > MagicMirror Pod

// Data Flow
Backend Pod <> RDS MySQL
Backend Pod > S3 Storage
Backend Pod > SQS Queue: send jobs
Worker Pod <-- SQS Queue: process jobs
Worker Pod > RDS MySQL

// Container Registry
EKS Cluster > ECR: pull images

// Monitoring Flow
EKS Cluster --> CloudWatch: logs & metrics
EKS Cluster --> Prometheus: metrics
Prometheus > Grafana
CloudWatch > SNS Alerts

// Security
Secrets Manager --> EKS Cluster: inject secrets
IAM Roles > EKS Cluster
Security Groups > VPC
